# VitaLiQ - Landing Page

## Descripcion General 
**VitaliQ** 
El desarrollo del landing page como parte del proyecto tiene como objetivo principal el **promover habitos saludables** mediante la plataforma digital informativa ,atractiva y funcional.

Esta solucion busca sensiblizar a los usuarios sobre la importancia del binestar fisico y mental, presentando contenido de valor y una interfaz moderna e intuitiva.
La pagina aplica buenas practicas de desarrollo web, integrando **HTML5**,**CSS**,**JavaScript** para lograr una experiencia dinamica y responsiva.

## Segmento Objetivo
La solucion esta dirigida a **jovenes universitarios y profesionales** que buscan mejorar su bienestar fisico y mental a traves de herramientas tecnologicas que promueven habitos saludables de forma practica , accesible y visualmente atractiva.

----

## Principales Caracteristicas 
- Diseno accesible y adaptable a moviles , tablets y computadoras
- Navegación fluida y estructura limpia basada en HTML
- **Formulario de contacto funcional** para comunicación directa con el equipo
- Secciones informativas con enfoque en bienestar y salud
- Uso de **Git** y **GitFlow** para control de versiones y trabajo colaborativo
- Estilo visual moderno con tipografia legible y colores suaves.  
 
----

## Herramientas Utilizadas 
- **HTML5** – Estructura y contenido de la página
- **CSS** – Estilos visuales, diseño adaptable y animaciones
- **JavaScript** – Interactividad y validaciones
- **Git / GitHub** – Control de versiones y flujo de trabajo en equipo  
- **WebStorm** – Entorno de desarrollo
